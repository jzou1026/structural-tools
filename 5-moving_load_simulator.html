What this app calculates

This app complements the design spreadsheet. The spreadsheet performs resistance and code-check calculations. This app performs the missing structural-analysis task: it moves vehicle axle loads along a continuous 1D girder model and envelopes MT and VT.

Step 1 — CSA S6-14 Cl. 3.8.1: Treat the selected traffic model as a live-load model within the Section 3 live-load framework.

Step 2 — CSA S6-14 Cl. 3.8.3.1.1: Represent CL-W loading conceptually as either a CL-W Truck case or a CL-W Lane Load case.

Step 3 — CSA S6-14 Cls. 3.8.3.1.2 and 3.8.3.1.3: Use the user-entered axle pattern and optional 9.0 kN/m lane UDL as the longitudinal line loads for the girder-line analysis.

Step 4 — CSA S6-14 Cl. 3.8.4.5: Apply dynamic load allowance through the user-entered DLA field. The current app multiplies the selected axle-load analysis by 1 + DLA when the user includes it.

Step 5 — CSA S6-14 Section 5 / Cl. 5.6: Use the user-entered truck load fraction FT to scale the 1D line-girder results into girder design effects where the simplified method is applicable.

Step 6 — Kassimali Ch. 9: Move the series of concentrated loads across the beam and envelope shear and bending moment responses at the output stations.

Step 7 — Kassimali Ch. 17: Solve each vehicle position using a span-level matrix stiffness formulation with rotational support degrees of freedom and fixed-end moments.

Limitations and user responsibilities
  • Not a full CSA S6 compliance checker: The app computes 1D analysis effects only; it does not generate every CSA S6 Section 3 load case automatically.
  • FT is manual: The user must calculate or justify FT using CSA S6-14 Section 5 simplified distribution, a grillage model, finite-element analysis, or other approved analysis.
  • Lane UDL behaviour: Full-span lane UDL is applied by fixed-end moments. Moving partial UDL is approximated by concentrated loads over the selected mesh spacing.
  • No transverse distribution model: The program is a 1D girder-line model, not a deck, grillage, or shell finite-element model.
  • No resistance design: The program does not perform steel moment resistance, shear resistance, fatigue, shear stud, splice, bearing, stiffener, or serviceability code checks.
  • CSA S6-14 clauses should be reviewed: Confirm design lanes, multi-lane factors, DLA category, load combinations, and governing load case outside this app before using results in design.
